## 📌 Project Overview

This project is a pixel-perfect implementation of the provided Figma design as part of the Web Developer hiring task for OneLittleWeb.

The goal was to accurately convert the design into a fully responsive and production-ready Next.js application, maintaining strong attention to layout, spacing, typography, and responsiveness.

---

## ⚙️ Tech Stack

- Next.js (App Router)
- Tailwind CSS
- TypeScript

---

## 🎯 Key Features

- Pixel-perfect UI based on Figma design
- Fully responsive across mobile, tablet, and desktop
- Clean and reusable component structure
- Optimized images and performance
- Semantic HTML for accessibility

---

## 📱 Responsiveness

Carefully implemented responsiveness to match the design intent across:

- Mobile devices
- Tablets
- Desktop screens

---

## 🧠 Implementation Approach

- Broke down the Figma design into reusable components
- Focused on consistent spacing and typography
- Used Tailwind utility classes for rapid and precise styling
- Ensured alignment and proportions match the design closely

---

Approximately **1 night (~6–8 hours)** of focused work.
